Boolean Algebra Laws

De Morgan's Theorems — Most Tested!

Theorem 1: A · B = Ā + B̄  (NAND = NOT-OR)
Theorem 2: A + B = Ā · B̄  (NOR = NOT-AND)

Memory trick: "Break the bar, change the operator." Invert each variable and flip AND↔OR.

Key Identities to Memorize

  • Idempotent: A + A = A, A · A = A
  • Complement: A + Ā = 1, A · Ā = 0
  • Absorption: A + AB = A, A(A+B) = A
  • Consensus: AB + ĀC + BC = AB + ĀC
  • Double negation: A = A
  • Distributive: A(B+C) = AB + AC
⚠️

Exam Trap: NAND and NOR are called "universal gates" — any Boolean function can be built using only NAND gates OR only NOR gates. This often appears as a direct MCQ.

🧠 Quick Check

What is the output of XOR when both inputs are the same (e.g., A=1, B=1)?

A) 1
B) 0 ✓
C) Undefined
XOR outputs 1 only when inputs are DIFFERENT (1⊕0=1, 0⊕1=1). When both are the same (0⊕0=0, 1⊕1=0), output is 0. XOR = "exclusive" or — detecting difference.

Apply De Morgan's: simplify NOT(A AND B)

A) NOT-A AND NOT-B
B) NOT-A OR NOT-B ✓
C) A OR B
De Morgan's Theorem 1: NOT(A·B) = NOT-A + NOT-B. "Break the bar, flip the operator": invert each variable and change AND to OR.

K-Maps & Boolean Simplification

Karnaugh Maps are the visual tool for minimizing Boolean expressions — reducing gate count in real chip designs.

K-Map Rules

How to Use a K-Map

  • Step 1: Fill the K-map with 1s from the truth table (minterms).
  • Step 2: Group adjacent 1s in powers of 2 (1, 2, 4, 8). Groups can wrap around edges!
  • Step 3: Make groups as large as possible.
  • Step 4: Each group gives one product term. Variables that change across the group are eliminated.
  • Step 5: OR all the product terms → minimized SOP expression.
2-Variable K-Map Example

F = Σm(1,2,3) — Simplify

B=0B=1
A=001
A=111

Group the three 1s: pair (A=0,B=1) and (A=1,B=1) → gives B. Pair (A=1,B=0) and (A=1,B=1) → gives A.

Result: F = A + B

Compared to raw SOP: ĀB + AB̄ + AB → K-map reduces it to just A + B (OR gate!)

💡

Don't-Care conditions (X or d): Minterms where output can be 0 or 1. You CAN include them in groups to make larger groups (better simplification), but you don't HAVE to.

SOP vs POS Forms

  • SOP (Sum of Products): Group the 1s → AND terms ORed together. e.g., AB + ĀC
  • POS (Product of Sums): Group the 0s → OR terms ANDed together. Used when fewer 0s.
  • Minterm: Row where output = 1 (used in SOP)
  • Maxterm: Row where output = 0 (used in POS)

🧠 Quick Check

In a K-map, groups must always be in sizes of:

A) Powers of 2 (1,2,4,8...) ✓
B) Any size
C) Multiples of 3
K-map groups must be powers of 2: 1, 2, 4, 8, 16... Larger groups eliminate more variables and give simpler expressions. A group of 4 eliminates 2 variables; a group of 8 eliminates 3.

Flip-Flops & Sequential Circuits

Sequential circuits have memory — output depends on current input AND past history. These implement registers, counters, and state machines.

The 4 Flip-Flop Types
Flip-FlopInputsBehaviorKey Use
SR (Set-Reset)S, RS=1→Q=1, R=1→Q=0, S=R=1→FORBIDDENBasic latch; S=R=1 is invalid state
D (Data)DQ(next) = D on clock edgeMost common; registers, memory cells
JKJ, KJ=K=1→Toggle; else like SR but no forbidden stateFixes SR's forbidden state issue
T (Toggle)TT=1→toggle; T=0→holdCounters

D Flip-Flop — Most Important for VLSI

DClock EdgeQ(next)
0↑ Rising0
1↑ Rising1

D flip-flop captures input D on the rising clock edge and holds it until the next clock edge. This is how registers in CPUs store data — it's the building block of all sequential logic.

Finite State Machine (FSM)

  • Moore FSM: Output depends only on current state. Simpler, more stable outputs.
  • Mealy FSM: Output depends on current state AND inputs. Faster response, fewer states.
  • States: Represent memory/conditions of the system
  • Transitions: Move between states based on inputs
  • n flip-flops can represent 2ⁿ states
⚠️

Setup & Hold Time: Setup time = data must be stable BEFORE clock edge. Hold time = data must remain stable AFTER clock edge. Violating these causes metastability — critical in VLSI timing analysis!

🧠 Quick Check

Which flip-flop is formed when J=K are tied together?

A) SR Flip-Flop
B) D Flip-Flop
C) T Flip-Flop ✓
When J and K inputs are tied together as a single input T, the JK flip-flop becomes a T (Toggle) flip-flop. When T=1 (J=K=1), the output toggles on each clock edge — perfect for binary counters.

CMOS Fundamentals

CMOS is the dominant technology in modern chips — from your phone's processor to the ICs designed at ULKASEMI. Understanding CMOS logic is essential for IC Physical Design roles.

CMOS Inverter — The Fundamental Building Block

How It Works

  • Input = 0 (LOW): PMOS turns ON (pulls output to VDD = 1), NMOS turns OFF
  • Input = 1 (HIGH): NMOS turns ON (pulls output to GND = 0), PMOS turns OFF
  • Key property: In steady state, never both ON simultaneously → no direct path from VDD to GND → no static power dissipation!
Output: Vout = NOT(Vin) — perfect inverter

CMOS NAND Gate (2-input)

  • Pull-down (NMOS): 2 NMOS in SERIES → output LOW only when both A=1 AND B=1
  • Pull-up (PMOS): 2 PMOS in PARALLEL → output HIGH when A=0 OR B=0
  • Total: 4 transistors for 2-input NAND
  • n-input NAND: n NMOS in series + n PMOS in parallel = 2n transistors
💡

Rule: PDN (NMOS) mirrors the pull-up logic with duality: SERIES in PDN = PARALLEL in PUN and vice versa.

MOSFET Operating Regions
RegionCondition (NMOS)Behavior
Cutoff (OFF)Vgs < VthNo current flows — transistor OFF (open switch)
Linear (Triode)Vgs > Vth AND Vds < Vgs−VthActs like resistor — ID proportional to Vds
SaturationVgs > Vth AND Vds ≥ Vgs−VthConstant current — used in analog amplifiers

🧠 Quick Check

In CMOS logic, which transistor type forms the Pull-Down Network (PDN)?

A) NMOS ✓
B) PMOS
C) Both equally
NMOS transistors form the Pull-Down Network (PDN) because they turn ON with HIGH gate voltage — pulling output down to GND. PMOS forms the Pull-Up Network (PUN), connecting output to VDD when gate is LOW. This complementary action is the "C" in CMOS.

Each Stage Explained

  • RTL (Register Transfer Level): Behavioral description in Verilog/VHDL. Describes WHAT the circuit does.
  • Logic Synthesis: Converts RTL to gate-level netlist using a standard cell library. Tools: Synopsys Design Compiler.
  • Floorplanning: Define die size, macro placement (memories, IPs), power grid planning.
  • Placement: Place standard cells in rows. Optimize for timing, power, and area.
  • CTS (Clock Tree Synthesis): Build balanced clock distribution to minimize skew.
  • Routing: Connect all wires (metal layers) while respecting design rules.
  • Signoff: STA (Static Timing), DRC, LVS checks before sending to foundry (tape-out).
Physical Design Checks

⚠️

Exam tip: Clock skew = difference in clock arrival time at different flip-flops. CTS minimizes skew. High skew = timing violations. Zero skew is ideal but unachievable in practice.

Timing Analysis & Power

Two critical metrics that every IC Physical Design engineer must master. Power and timing directly determine if a chip passes or fails signoff.

Timing Fundamentals

Setup & Hold Time

  • Setup time (Tsu): Data must be STABLE this long BEFORE the clock edge
  • Hold time (Th): Data must be STABLE this long AFTER the clock edge
  • Propagation delay (Tpd): Time for output to change after input changes
  • Clock period constraint: Tclk ≥ Tpd(logic) + Tsu + Clock-skew
Max Freq: fmax = 1 / (Tpd + Tsu + Tskew)

Critical Path

The longest timing path in the design — it determines the maximum operating frequency. To increase frequency, you must reduce the critical path delay by:

  • Logic restructuring (pipelining — add registers to split long path)
  • Gate sizing (use faster, larger cells)
  • Better placement (reduce wire delay)
Power Dissipation in CMOS
TypeCauseFormulaReduces by

Static (Leakage)Subthreshold leakage currentP = Ileak × VDDHigher Vth, power gating

💡

Power-Performance Tradeoff: Reducing VDD by 2× reduces dynamic power by 4× (quadratic!) but also slows the circuit. Modern chips use DVFS (Dynamic Voltage and Frequency Scaling) to balance both.

🧠 Quick Check

Which type of power is dominant in modern high-frequency CMOS chips?

A) Dynamic power ✓
B) Static leakage power
C) Short-circuit power
Dynamic power (P = αCV²f) dominates in high-frequency digital chips. However, as transistors shrink to nanometer scales, static leakage power becomes increasingly significant — it's now a major concern in sub-20nm designs even at idle.

Semiconductors & PN Junction

The physics behind every transistor ever built. Silicon's remarkable properties make modern electronics possible.

Semiconductor Basics

Intrinsic vs Extrinsic

  • Intrinsic semiconductor: Pure silicon — very few carriers at room temperature. ni = 1.5×10¹⁰ /cm³ for Si at 300K.
  • N-type doping (donor): Add Group V elements (P, As, Sb) → extra electrons → majority = electrons
  • P-type doping (acceptor): Add Group III elements (B, Al) → holes → majority = holes
  • Fermi level: Moves toward conduction band for N-type, toward valence band for P-type
Mass action: n · p = ni² (always valid in thermal equilibrium)
PN Junction Diode

Depletion Region & Built-in Voltage

When P and N types are joined, electrons diffuse to P-side and holes to N-side, creating a depletion region with a built-in electric field opposing further diffusion.

  • Built-in voltage (Vbi): ~0.6–0.7V for silicon
  • Forward bias: Apply +V to P, narrows depletion region → current flows (diode ON)
  • Reverse bias: Apply +V to N, widens depletion region → very small reverse current (diode OFF)
  • Breakdown voltage: At high reverse voltage, Zener/avalanche breakdown occurs
Diode eq: I = Is(e^(V/VT) − 1)  where VT ≈ 26mV at 300K

BJT & MOSFET Transistors

The two dominant transistor types — BJT in analog/power, MOSFET dominating digital CMOS. Both appear heavily on IC design exams.

BJT (Bipolar Junction Transistor)
RegionCondition (NPN)Behavior
CutoffVBE < 0.6V, VBC < 0OFF — no current flows
Active (Forward)VBE ≈ 0.6V, VBC < 0Amplifier mode — IC = β·IB
SaturationVBE > 0, VBC > 0Both junctions forward biased — switch ON
Reverse ActiveVBE < 0, VBC > 0Reversed role — low gain, rarely used

BJT Key Relationships

β (hFE): IC = β · IB  (β typically 50–300)
α: IC = α · IE  (α = β/(β+1) ≈ 0.99)
KCL: IE = IC + IB

BJT is a current-controlled device — base current controls collector current.

MOSFET vs BJT Comparison
PropertyMOSFETBJT
ControlVoltage-controlled (Vgs)Current-controlled (IB)
Input impedanceVery high (gate is insulated)Low (base draws current)
Power consumptionVery low staticHigher static

ScalabilityExcellent (nm scale)Limited

⚠️

Threshold voltage (Vth): The gate voltage at which MOSFET starts to conduct. Vth depends on doping, oxide thickness, and substrate bias. Body effect (VSB > 0) increases Vth in NMOS.

🧠 Quick Check

An NPN BJT has IB = 50μA and β = 100. What is the collector current IC?

A) 50μA
B) 5mA ✓
C) 500mA
IC = β × IB = 100 × 50μA = 5000μA = 5mA. This is the fundamental BJT current gain relationship. β (or hFE) is the DC current gain of the transistor in the active region.

Op-Amps & RC Circuits

Operational amplifiers and passive RC networks are the workhorses of analog design — appearing in filters, ADCs, and mixed-signal IC interfaces.

Ideal Op-Amp Properties

The Golden Rules

  • Infinite open-loop gain (A → ∞)
  • Infinite input impedance → zero input current
  • Zero output impedance
  • Infinite bandwidth
  • Virtual short: V+ = V− (when negative feedback is applied)
  • Virtual open: No current flows into input terminals
Key Op-Amp Configurations
ConfigurationGain FormulaKey Feature
Inverting AmplifierAv = −Rf/RinOutput inverted, 180° phase shift
Non-Inverting AmplifierAv = 1 + Rf/R1High input impedance, same phase
Voltage Follower (Buffer)Av = 1Rf=0, R1=∞; impedance matching
Summing AmplifierVout = −Rf(V1/R1 + V2/R2)Inverting adder for multiple inputs
DifferentiatorVout = −RC · dVin/dtC input, R feedback
IntegratorVout = −(1/RC)∫Vin·dtR input, C feedback
ComparatorNo feedbackOutput = ±Vsat based on V+−V−
RC Circuit Time Constants

RC Charging & Discharging

Time const: τ = R × C  (in seconds)
Charging: V(t) = Vs(1 − e^(−t/τ))
Discharging: V(t) = V0 · e^(−t/τ)
  • At t = τ: capacitor charged to 63.2% of final voltage
  • At t = 5τ: fully charged (99.3%) — practical "done"
  • Cutoff frequency: fc = 1/(2πRC) — frequency where gain drops 3dB
  • Low-pass filter: R in series, C to ground — passes low freq, blocks high
  • High-pass filter: C in series, R to ground — passes high freq, blocks low
✅

Bandwidth-Gain Tradeoff (GBW): For a real op-amp, Gain × Bandwidth = constant. If you increase gain, bandwidth decreases proportionally. This is the Gain-Bandwidth Product (GBP).

🧠 Quick Check

An inverting amplifier has Rin = 10kΩ and Rf = 100kΩ. What is the voltage gain?

A) +10
B) −10 ✓
C) −0.1
For an inverting amplifier: Av = −Rf/Rin = −100kΩ/10kΩ = −10. The negative sign indicates phase inversion (output is inverted). The magnitude of gain is 10, meaning the output is 10× amplified and flipped.

In an RC circuit with R = 1kΩ and C = 1μF, what is the time constant τ?

A) 1ms ✓
B) 1μs
C) 1s
τ = R × C = 1×10³ × 1×10⁻⁶ = 1×10⁻³ = 1ms. After 1ms, the capacitor reaches 63.2% of the supply voltage. After 5ms (5τ), it's considered fully charged (99.3%).
