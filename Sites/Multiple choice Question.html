🔢 Python Essentials

List Operations

  • lst[-1] → last element
  • lst[1:3] → index 1,2 only
  • lst[::-1] → reversed list
  • len(lst) → length
  • lst.append(x) → add to end
  • lst.pop() → remove last

String Tricks

  • "abc"[1] → 'b'
  • "ab" * 3 → 'ababab'
  • "A".lower() → 'a'
  • "a,b".split(",") → ['a','b']
  • f"{x:.2f}" → 2 decimal float

Common Gotchas

  • range(5) = 0,1,2,3,4 (not 5)
  • 5//2 = 2, 5%2 = 1
  • bool(0) = False, bool("") = False
  • Mutable default args are shared!
  • is checks identity, == checks value

Complexity Cheatsheet

  • List access: O(1)
  • List search: O(n)
  • Dict lookup: O(1) avg
  • Sorting: O(n log n)
  • Binary search: O(log n)

⚙️ C/C++ Quick Reference

Pointers

  • int *p = &x → p holds address of x
  • *p → dereference (value at address)
  • p++ → next int address (+4 bytes)
  • Array name = pointer to first element
  • NULL pointer = address 0

Operators & Precedence

  • ++i pre-increment (increment first)
  • i++ post-increment (use then increment)
  • a & b bitwise AND
  • a | b bitwise OR
  • a ^ b bitwise XOR
  • a << 1 = a × 2

Data Types (32-bit)

  • int: 4 bytes, −2³¹ to 2³¹−1
  • char: 1 byte
  • float: 4 bytes
  • double: 8 bytes
  • short: 2 bytes
  • Integer overflow wraps around!

Common Output Traps

  • printf("%d", 5/2) → 2 (integer div)
  • printf("%f", 5/2) → 2.0 (still int div!)
  • printf("%f", 5.0/2) → 2.5
  • Uninitialized vars have garbage values

🧠 Aptitude Formulas

Number Series Patterns

  • Arithmetic: constant difference
  • Geometric: constant ratio
  • Fibonacci: each = sum of prev two
  • Squares: 1,4,9,16,25,36...
  • Primes: 2,3,5,7,11,13,17...

Percentage & Ratio

  • % change = (new−old)/old × 100
  • If A:B = 3:4 and total=70 → A=30, B=40
  • Simple Interest = P×R×T/100
  • 20% of 150 = 30
  • X is 25% more than Y → X = 1.25Y

Work & Time

  • A does in n days → rate = 1/n per day
  • A+B together: 1/a + 1/b per day
  • Combined time = ab/(a+b)
  • Speed = Distance / Time
  • Avg speed = 2v₁v₂/(v₁+v₂) for equal dist

Probability & Logic

  • P(A) = favorable / total outcomes
  • P(A∪B) = P(A)+P(B)−P(A∩B)
  • All A are B + All B are C → All A are C
  • Deductive: general → specific
  • Inductive: specific → general
